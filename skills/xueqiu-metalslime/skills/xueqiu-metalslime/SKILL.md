# 雪球 metalslime 框架

这个 skill 更偏交易与风格切换，不适合拿来装成严格意义上的传统价值投资。

## 何时使用

- 用户问当前市场主线怎么切
- 用户要在互联网、黄金、周期、新能源、医药等方向之间切换
- 用户需要一种更贴近盘面与情绪的“先活下来，再吃主线”框架
- 用户想把宏观叙事翻译成交易优先级

## 先做什么

先读 `references/profile.md`。如果要给来源，再读 `references/sources.md`。

## 分析顺序

1. 先识别市场在奖励什么。
   红利、成长、周期、主题、港股互联网，当前谁拿到资金。
2. 再看宏观叙事是否切换。
   通胀、利率、地缘、政策、商品价格、居民部门预期是否变化。
3. 再看能不能交易。
   逻辑再好，不在当前主线里，也可能暂时赚不到钱。
4. 再做仓位排序。
   先主线，再次主线，再观察仓。
5. 最后强调情绪管理。
   交易时情绪脱离市场，才能执行；被市场牵着走就容易反复打脸。

## 输出格式

- `当前主线`
- `宏观叙事`
- `交易优先级`
- `仓位建议`
- `情绪与纪律`
- `失效条件`

## 风格要求

- 可以犀利，但要有逻辑
- 允许快速切换，但要解释为什么切
- 不迷信长期不动，也不鼓励无脑高频
- 先判断“钱往哪去”，再谈理想世界

## 不该怎么用

- 不要伪装成纯基本面研究
- 不要省略风格与资金面
- 不要给“永远正确”的大叙事
