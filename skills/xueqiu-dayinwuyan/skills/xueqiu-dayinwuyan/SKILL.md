# 雪球大隐无言框架

这个 skill 适合处理创新药、管线、BD、海外上市预期和高成长估值问题。

## 何时使用

- 用户在看荣昌生物、康方生物、创新药 BD 或海外临床逻辑
- 用户需要把“故事股”拆成收入、适应症、里程碑与估值
- 用户需要一种能忍受高波动、但要求持续验证的数据化成长框架

## 先做什么

先读 `references/profile.md`。如果要给出处，再读 `references/sources.md`。

## 分析顺序

1. 先列产品和适应症。
   不要只讲公司，不讲管线。
2. 再列未来 2 到 5 年关键催化。
   临床结果、BD、申报、获批、医保、销售放量要分别说。
3. 再给增长假设。
   先给收入，再谈 PS 或利润。
4. 再讨论市场为什么可能看错。
   通常是把青年期创新药按成熟药企估值。
5. 最后给持有逻辑与证伪点。
   什么情况下继续拿，什么情况下减仓。

## 输出格式

- `核心管线`
- `关键催化`
- `收入与估值假设`
- `市场预期差`
- `持有前提`
- `证伪条件`

## 风格要求

- 敢给中长期假设，但要写清假设链条
- 更重产品与商业化，而不是 K 线
- 把波动视为常态，把证伪视为边界
- 允许乐观，但必须给出触发条件

## 不该怎么用

- 不要把它写成纯短线打板
- 不要只看 PE 忽略创新药早期公司的阶段性
- 不要在没有产品和临床信息时强行估值
