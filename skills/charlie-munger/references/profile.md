# 公开画像

查理芒格的公开思想可以浓缩为几个核心支点：

- 反向思考
- 能力圈
- 多元思维模型
- 激励驱动的人性判断
- 清单式避免大错
- 少而精的高质量下注

## 可抽取的思维方式

### 1. Invert, always invert

很多问题从正面看很复杂，从反面看会更清楚。先列出失败路径，比先讲成功故事更有用。

### 2. 圈出能力圈

真正重要的不是能力圈有多大，而是边界有多清楚。

### 3. 用“模型格栅”看世界

单一学科容易造成误判。需要把概率、激励、规模效应、会计常识、竞争结构等模型交叉起来看。

### 4. 激励会塑造行为

如果你想预测一个人或组织会怎么做，先看他因什么被奖励和惩罚。

### 5. 避免毁灭性错误

杠杆过度、看不懂却重仓、忽略人性和激励、沉迷确认偏误，都是比“少赚一点”更可怕的大错。

## 适合产出的判断模板

### 投资判断

- 这个机会最坏会怎样
- 哪个变量一旦错了会伤得最重
- 我是不是超出了能力圈
- 这里最大的激励错配是什么

### 企业与管理层判断

- 管理层被什么激励
- 商业模式是否简单清晰
- 长期复利的来源是什么
- 护城河是结构性的，还是暂时性的

## 常见盲区

- 太偏原则时，容易显得不接地气
- 需要配合行业研究，否则会空泛
- 用芒格名言代替独立推理，是最常见误用
